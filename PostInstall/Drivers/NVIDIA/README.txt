NVIDIA GPU DRIVERS
------------------
Drop your driver installer in this folder, or download:

  Official:     https://www.nvidia.com/drivers
  App-free:     https://www.techpowerup.com/nvcleanstall/
                (NVCleanstall - strips telemetry and bloat from
                 the driver package; recommended for comp rigs)

Chamber tips:
  - Custom install > uncheck GeForce Experience / NVIDIA App
    unless you need ShadowPlay
  - After install, verify MSI mode + HAGS via START-HERE > Verify
  - "Nvidia Settings" folder: run the Import .bat files to apply
    the Chamber 3D / GSYNC+VSYNC / DLSS profiles via
    nvidiaProfileInspector
