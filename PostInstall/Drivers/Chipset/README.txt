CHIPSET DRIVERS - INSTALL THESE FIRST
-------------------------------------
  AMD (X670/B650/X870 etc):
    https://www.amd.com/en/support  > Chipsets

  Intel:
    https://www.intel.com/content/www/us/en/support/detect.html

Why first: chipset drivers control CPU power management and
core scheduling (critical on X3D and dual-CCD CPUs). Install,
reboot, THEN install GPU drivers.
